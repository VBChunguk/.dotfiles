# Dotfiles

```sh
cd
git clone https://github.com/tirr-c/.dotfiles
ln -sf .dotfiles/.zshrc
ln -sf .dotfiles/.vimrc
ln -sf .dotfiles/.tmux.conf
ln -sf .dotfiles/.npmrc
```

Or,

```sh
cd
git clone https://github.com/tirr-c/.dotfiles
./.dotfiles/install.sh
```
